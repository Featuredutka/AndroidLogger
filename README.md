# AndroidLogger

Android application for logging data from your phone sensors.
#### Supported sensors:
  * Accelerometer
  * Gyroscope
  * Magnetometer
#### Capabilities: 
- [x]  Reading and showing sensors data
- [x]  Switching sensor reading on/off
- [x]  Writing data to .csv files
- [ ]  Plotting real-time data
